# Chemy

Мы создаем многопользовательскую игру, в которой используемся механика крафта путем смешивания елементов. В этой игре есть множество веток развития, игра развивается как многомерный граф. Узлы — это компоненты, некоторые ноды состоят из множества объединений. Задача игроков — раскрыть все элементы. 

## Задача
Нужно создать читабельную для авторов игры таблицу крафта, редактируя которую автор сможет создать многомерный граф рецептов.
